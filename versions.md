# Versions

## V0

Status: Active

V0 establishes the initial Engineering OS structure.

Includes:

- Vision
- Principles
- Collaboration rules
- Roadmap
- Prompt library
- Decision log
- Experiments area

Core operating components:

- Daily Capture
- Weekly Snapshot
- Current Thinking
- Staff Chief of Staff Agent

## Release rule

Each version change must include:

- Problem
- Proposal
- Impact
- Action

## Evaluation rule

Do not change the system immediately just because a better idea appears.

Run the current version long enough to observe a real problem first.
