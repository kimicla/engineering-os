# Engineering OS

Engineering OS is a personal operating system for leading complex engineering work.

It is not a knowledge base.
It is not a task manager.
It is not a productivity aesthetic.

Its purpose is to continuously improve how I think, prioritize, make decisions, and lead engineering projects.

Project knowledge belongs to individual projects.
Engineering OS belongs to me.

## What this system optimizes for

- Better engineering judgment
- Clearer prioritization
- Faster decision making
- Lower cognitive load
- More consistent project leadership
- Less time managing information

## Core loop

```text
Daily Capture
  -> Weekly Snapshot
  -> Current Thinking
  -> Daily Focus
  -> Execution
  -> New Evidence
```

The goal is not to document everything.
The goal is to keep my current judgment aligned with reality.
