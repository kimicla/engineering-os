# Weekly Review Prompt

## Prompt

```text
Weekly Review.

Read this week's Daily Notes, Current Thinking, and NEXT list.

Generate:

1. Weekly Summary
2. Major Decisions
3. Open Questions
4. Recommended Focus for Next Week
5. Should Current Thinking be updated? If yes, what exact lines should change?

Keep concise.
Do not summarize every task.
Focus on what changed, what became clear, and what still needs a decision.
```

## Quality bar

A good Weekly Snapshot should answer:

- What changed this week?
- What decisions were made?
- What is still unresolved?
- What should matter next week?
- Does Current Thinking need to change?

## Anti-patterns

- Listing every task completed.
- Writing a status report instead of a thinking update.
- Creating more than three next-week focus areas.
- Updating Current Thinking when judgment did not change.
