# Current Thinking Prompt

## Prompt

```text
Refresh Current Thinking.

Use the latest Weekly Snapshot and the existing Current Thinking.

Update only if judgment changed.
Do not rewrite for style only.

Generate:

1. Current Bet
2. Current Recommendation
3. Top 3 Active Initiatives
4. Top 3 Risks
5. Decisions Needed Soon
6. Ignore For Now

Keep under 500 words.
Prefer clarity over completeness.
```

## Purpose

Current Thinking represents what I currently believe about the project.

It is not a summary.
It is not a status report.
It is not a task list.

It is the strategic source for daily focus.

## Update rule

Only update Current Thinking when one of these changes:

- The current bet changed.
- The main constraint changed.
- A major decision was made.
- A top risk changed.
- An initiative became irrelevant or newly important.

Do not update it just because new notes exist.
